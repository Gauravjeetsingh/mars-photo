Welcome to the Mars Photo API

This API provides access to photos taken by NASA's Mars rovers: Curiosity, Perseverance, Opportunity, and Spirit.

All endpoints return JSON data and support CORS for cross-origin requests.

GET /api/v1/rovers
Get a list of all available Mars rovers with their mission information.
Example Request
GET /api/v1/rovers

Response

{ "rovers": [ { "id": 5, "name": "Curiosity", "landing_date": "2012-08-06", "launch_date": "2011-11-26", "status": "active", "max_sol": 4234, "max_date": "2024-01-15", "total_photos": 500000, "cameras": [...] } ] }
GET /api/v1/rovers/:rover_id
Get detailed information about a specific rover. Valid rover IDs: curiosity, perseverance, opportunity, spirit.

Path Parameters

  • rover_id string Required
    The rover identifier (curiosity, perseverance, opportunity, or spirit)
Example Request
GET /api/v1/rovers/curiosity

Response

{ "rover": { "id": 5, "name": "Curiosity", "landing_date": "2012-08-06", "launch_date": "2011-11-26", "status": "active", "max_sol": 4234, "max_date": "2024-01-15", "total_photos": 500000, "cameras": [ { "name": "FHAZ", "full_name": "Front Hazard Avoidance Camera" }, ... ] } }
GET /api/v1/rovers/:rover_id/photos
Get photos taken by a specific rover. You must provide either sol or earth_date parameter.

Path Parameters

  • rover_id string Required
    The rover identifier

Query Parameters

  • sol number Optional
    Martian sol (day) number. Either sol or earth_date is required.
  • earth_date string (YYYY-MM-DD) Optional
    Earth date. Either sol or earth_date is required.
  • camera string Optional
    Filter by camera name (e.g., FHAZ, NAVCAM, MAST)
  • page number Optional
    Page number for pagination (default: 0)
  • per_page number Optional
    Results per page (default: 25)
Example Requests
GET /api/v1/rovers/curiosity/photos?sol=1000 GET /api/v1/rovers/curiosity/photos?earth_date=2015-06-03&camera=FHAZ GET /api/v1/rovers/perseverance/photos?sol=500&page=0&per_page=25
GET /api/v1/rovers/:rover_id/latest_photos
Get the latest photos taken by a specific rover (from the most recent sol).

Path Parameters

  • rover_id string Required
    The rover identifier

Query Parameters

  • camera string Optional
    Filter by camera name
  • page number Optional
    Page number for pagination (default: 0)
  • per_page number Optional
    Results per page (default: 25)
Example Request
GET /api/v1/rovers/curiosity/latest_photos?camera=MAST
GET /api/v1/manifests/:rover_id
Get the mission manifest for a rover, including photos grouped by sol.

Path Parameters

  • rover_id string Required
    The rover identifier
Example Request
GET /api/v1/manifests/curiosity

Response

{ "photo_manifest": { "name": "Curiosity", "landing_date": "2012-08-06", "launch_date": "2011-11-26", "status": "active", "max_sol": 4234, "max_date": "2024-01-15", "total_photos": 500000, "photos": [ { "sol": 1000, "earth_date": "2015-05-30", "total_photos": 150, "cameras": ["FHAZ", "RHAZ", "NAVCAM"] }, ... ] } }
